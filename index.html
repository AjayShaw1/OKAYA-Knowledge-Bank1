📘 OKAYA CRM SOP (Standard Operating Procedure)

🔐 1. Login to CRM

URL: https://okayacarecrm.com

  • Enter your User ID and Password
  • Click Login
  • Dashboard shows Complaint Login, Product Registration, Lead Query etc.

🛠️ 2. Complaint Registration (Battery/Inverter/Nasaka)

  1. Click Complaint Login
  2. Enter Mobile Number
  3. If new number ➝ Click Add New Customer
  4. Fill details like name, address, serial number, DOP, remarks, etc.

Serial Number Formats:

  • Battery: 16 digits
  • Inverter: 17 digits
  • Nasaka: 13 digits
  • If unknown ➝ use dummy: ZZZZZZZZZZ

📩 3. Complaint View / Status

  1. Click Complaint View
  2. Enter complaint/mobile/serial number ➝ Click View
  3. Shows assigned engineer, visit date, repair/replacement, HO decision etc.
